- **Data Processed:** IP address, time of access, page accessed, browser type.

- **Key Feature (Dataminimization):** The Website is a **statically generated information service (React/Vite)**. **No** cookies are set, **no** analysis services (like Google Analytics) are used, and **no** data is stored in a database (like Firestore or local storage).
    

### Security (Cloudflare)

- **Data Processed:** IP address, browser information.
    
- **Purpose:** The Website utilizes Cloudflare's security functions to protect it from DDoS attacks and malicious traffic. Cloudflare acts here as a Content Delivery Network (CDN) and processes IP addresses to filter traffic (Art. 6 para. 1 lit. f GDPR).
